# Activity Manager REST API

This is a REST API for managing activities. It is built using the NestJS framework and uses a PostgreSQL database.

## Description

Technologies used:

- NestJS
- PostgreSQL
- MikroORM

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
